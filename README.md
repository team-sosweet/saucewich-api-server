# saucewich-api-server

Saucewich api server for the part of out-game. 

## Configure environments

```bash
$ pyenv virtualenv 3.7.3 saucewich-api-server
$ pyenv local saucewich-api-server
$ pip install -r requirements.txt
```

## Specifications

- Account Data
    - ID & PW
    - Nickname
    - Friends
    - Level / exp
    - point
    - money
    - KDR
    - W/L
    - Playtime
- Matchmaking
- Notice/Event System
